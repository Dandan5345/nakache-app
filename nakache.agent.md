# Nakache.app - Project Agent

## תיאור הפרויקט (Project Description)
פרויקט זה מיועד להיות דף נחיתה ואתר פורטפוליו לאפליקציות תחת המותג Nakache.app. 
האתר נועד להציג את האפליקציות השונות, לספק הסברים אודותיהן, ולהכיל דפים ייעודיים למסמכים משפטיים כמו תנאי שימוש (Terms of Use) ומדיניות פרטיות (Privacy Policy) לכל אפליקציה, כך שניתן יהיה לקשר אליהם ישירות מחנויות האפליקציות.

## האפליקציות בפרויקט:
1. **TripEase**: אפליקציה לניהול ויצירת טיולים הכל במקום אחד (טיסות, מלונות, לו"ז יומי, דרכונים, אישורי הזמנה, ושיתוף עם חברים/משפחה). הפרויקט תחת פיתוח, ולכן מוצג ללא קישור כרגע.
2. **CashFlowM**: אפליקציה לניהול הוצאות והכנסות, עוקב תיק השקעות וחיסכונות, כולל שיתוף פיננסי עם חברים. אפליקציה זו זמינה בחנות האפליקציות של אפל.

## מבנה הפרויקט (Architecture / Stack)
- טכנולוגיות: HTML5, CSS3 הונילי, וטיפה JavaScript אם צריך לאינטראקציות.
- עיצוב נקי, מודרני (Modern UI/Glassmorphism) ומותאם למובייל (Responsive Design).
- תמיכה מלאה ב-RTL (ימין-לשמאל) עבור קהל יעד דובר עברית.

## שלבים לפיתוח (Roadmap)
1. יצירת דף הבית קטלוגי (`index.html`) עם Hero section יפה וכרטיסיות עבור כל אפליקציה.
2. עיצוב בעזרת קובץ `styles.css`.
3. יצירת תבנית לדפי מדיניות פרטיות ותנאי שימוש (Legal pages).
4. הוספת תמונות מהאפליקציות (Assets) בהתאם להוראות המשתמש.

## הנחיות למודל (Agent Instructions)
כאשר המשתמש מבקש לעשות שינויים באתר, יש לשמור על עיצוב התואם לשימוש בטלפונים חכמים, לשמור על אסתטיקה גבוהה ולהשתמש באנימציות CSS עדינות במידת האפשר.
